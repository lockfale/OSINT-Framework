# OSINT Framework - Maintainer Recruitment

## Current Status

The OSINT Framework has grown significantly:
- **9.4k+ stars** on GitHub
- **1,360+ tools** in database
- **1.6k+ forks**
- **Active community**

However:
- ⚠️ **32 open pull requests** (backlog)
- ⚠️ **57 open issues**
- ⚠️ **Single maintainer** (@jnordine)

**We need help!**

## Maintainer Roles

### 1. Link Validator (1-2 people)
**Time**: 2-4 hours/week

**Responsibilities**:
- Run weekly link validation
- Create issues for broken links
- Merge link removal/replacement PRs
- Maintain health reports

**Skills**: Git/GitHub, Node.js scripts, attention to detail

### 2. PR Reviewer (2-3 people)
**Time**: 3-5 hours/week

**Responsibilities**:
- Review tool submissions
- Verify categorization
- Check for duplicates
- Merge approved PRs

**Skills**: Git workflow, OSINT knowledge, JSON editing

### 3. Issue Triage (1-2 people)
**Time**: 2-3 hours/week

**Responsibilities**:
- Label and prioritize issues
- Close duplicates/spam
- Request additional info
- Convert issues to PRs

**Skills**: GitHub management, communication

### 4. Technical Lead (1 person)
**Time**: 5-8 hours/week

**Responsibilities**:
- Oversee modernization
- Review architectural changes
- Manage releases
- Coordinate maintainers
- Set roadmap

**Skills**: JavaScript/Node.js, D3.js, DevOps, leadership

## How to Apply

1. **Express Interest**: Open issue titled "Maintainer Application: [Role]"
2. **Demonstrate**: Submit 2-3 quality PRs
3. **Interview**: Brief discussion (Technical Lead only)

### Application Template

```markdown
## Maintainer Application

**GitHub**: @your-username

**References**: Other OSS projects (optional)
```
